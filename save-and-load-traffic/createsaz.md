# Create a Session Archive Zip (SAZ) Traffic archive

1. Start Fiddler. 

2. Use your client or browser to generate some HTTP/HTTPS traffic.

3. Click **File > Save > All Sessions...** 

4. Save the traffic to a **.SAZ** file.
